About the Project

About Us

This website was created by Judi Bush, Matt Stecklow, and Luna Yee. Its primary purpose is to provide access to interesting and potentially useful information, and its secondary purpose is to practice web development and proper coding practices.

Attribution

This website was built using the HTML5 Boilerplate template (v7.2.0), and incorporates images from Subtle Patterns by Toptal. Database integration is accomplished using PostgreSQL.

The data used on the site are from the ICPSR database; more details are available on the About the Data page.
